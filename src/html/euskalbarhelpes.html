Consultas en Euskalbar

Las consultas se pueden realizar de diversas maneras:

  • Pulsando los botones de la barra de herramientas

    De esta manera se consulta un diccionario por vez.

  • Pulsando la tecla Enter

    De esta manera se consultan, de forma simultánea, los diccionarios marcados en el apartado "Opciones", cada uno en una pestaña.

  • Pulsando las teclas Shift+Enter

    De esta manera, en el caso de tener seleccionado el idioma Español, se consultan Euskalterm, Elhuyar y 3000 de forma simultánea y en la misma pestaña; en caso de tener seleccionado el idioma Ingl&eecute;s, se consultan Euskalterm y el diccionario Morris.

  • Pulsando las teclas Ctrl+Enter

    De esta manera se consultan diccionarios de sinónimos de forma simultánea y en la misma pestaña.

  • Pulsando el botón derecho del ratón

    Seleccionando un texto en una página web, pulsando el botón derecho del ratón y eligiendo el diccionario a consultar en el submenú "Euskalbar".

Opciones

En este apartado se pueden determinar las siguientes opciones:

  • Mostrar/esconder el menú de diccionarios

    Muestra o esconde el menú de diccionarios, situado entre las opciones "Herramientas" y "Ayuda" del menú principal de Firefox.

  • Preferencias...

    • Idioma

      Establece el idioma predeterminado del botón de selección de idioma: eu >> es, es >> eu, eu >> en edo en >> eu.

    • Preferencias de las pestañas

      Establece el comportamiento de las pestañas: permite reutilizar las pestañas, y abrir las nuevas pestañas en el fondo.

    • Seleccione los diccionarios a consultar con la tecla Intro

    • Tiempo límite de las consultas combinadas (seg)

      Modifica el tiempo límite de consulta al usar Shift+Intro o Ctrl+Intro.

      Al usar este tipo de consulta, Euskalbar espera respuesta durante 5 segundos. Si transcurrido este tiempo no recibe respuesta, cierra la conexión.
      Este periodo de tiempo suele ser suficiente para recibir respuesta de los servidores donde están alojados los diccionarios, pero en algunos casos puede ocurrir que estos servidores necesiten más tiempo.

      Aviso: si se establece un tiempo límite demasiado corto, puede ocurrir que se reciba solamente una parte de la respuesta. Por ejemplo, si se consulta en Euskalterm la palabra "escuela" con un tiempo límite de 2 segundos, no se reciben todos los resultados.

    • Skins

      Permite cambiar el aspecto de las interfaces de Euskalbar.

  • Estadísticas

    Muestra un recuento de las búsquedas realizadas con cada uno de los diccionarios de la barra.

  • Mostrar/esconder los botones de la barra

    Pulsando con el botón derecho sobre la barra, existe la posibilidad de mostrar o esconder algunos de los diccionarios.
