Euskalbar searches

You can look up dictionaries in several ways:

  • Pressing the toolbar buttons

    In that way, a click means a look up in the clicked dictionary.

  • Pressing the Enter key

    In that way, the dictionaries chosen in the Options section will be looked up at the same time, each one in one tab.

  • Pressing Shift+Enter or Ctrl+Enter keys

    The dictionaries chosen in the Options section will be looked up at the same time, in the same tab.

  • Right-clicking

    Selecting text within a web page, doing right-click and selecting the desired dictionary under the submenu "Euskalbar".

Options

This section contains the following items:

  • Toggle dictionaries menu

    Toggles dictionaries menu. This menu is located between "Tools" and "Help" menus in the main Firefox toolbar.

  • Preferences...

    • Default language

      Sets the language used in the language selection button when Euskalbar starts: EU ‣ ES, ES ‣ EU, EU ‣ FR, FR ‣ EU, EU ‣ EN, EN ‣ EU or LA ‣ EU.

    • Tab preferences

      Preferences for tab behaviour: reuse tabas and open new tabs in background.

    • Skins

      Allows to change the skins for Euskalbar.

      Apart from the default Euskalbar skins, it is posible to use customized skins. Euskalbar's website offers several skins. You could also create your own skins based on those offered by Euskalbar, if you know how to manipulate CSS files.

    • GUI

      Offers two options to modify Euskalbar's GUI. They can also be modified using shortcuts:

      • Ctrl+Alt+H: Shows/hides the dictionaries menu
      • Ctrl+Alt+T: Shows/hides the context menu

    • Advanced query (Enter, Shift+Enter, Ctrl+Enter)

      Allows to select which dictionaries will be shown when performing advanced queries. 

    • Combined query options

      Allows to set some options of the combined queries (Shift+Enter, Ctrl+Enter).

      Query timeout (s): By default, when this queries are used, Euskalbar waits for 5 seconds to have a response. If there is no response, the connection is closed. This timeout use to be enough, but sometimes the servers where the dictionaries are hosted need more time to send the response. If the query timeout is too short, Euskalbar could receive a partial response. For instance, if you query Euskalterm with a 2 second timeout, you will get part of the whole response.

      Perform subqueries: Allows to perform subqueries in Elhuyar, 3000 and Batua dictionaries.

  • Statistics

    Shows a count of searches made with each of the bar dictionaries.

  • Show/hide bar buttons

    Richt-clicking in the bar shows a menu to toggle bar buttons.

Shortcuts

Euskalbar offers several shortcuts to facilitate the daily work:
  • Ctrl+Alt+A: Shows/hides Euskalbar
  • Ctrl+Alt+S: Focuses the textbox and, if it contains some text, selects it
  • Ctrl+Alt+Z: Toggles the language direction
  • Ctrl+Alt+H: Shows/hides the dictionaries menu
  • Ctrl+Alt+T: Shows/hides the context menu
