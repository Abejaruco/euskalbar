Euskalbar searches

You can look up dictionaries in several ways:

  • Pressing the toolbar buttons

    In that way, a click means a look up in the clicked dictionary.

  • Pressing the Enter key

    In that way, the dictionaries chosen in the Options section will be looked up at the same time, each one in one tab.

  • Pressing Shift+Enter or Ctrl+Enter keys

    The dictionaries chosen in the Options section will be looked up at the same time, in the same tab.

  • Right-clicking

    Selecting text within a web page, doing right-click and selecting the desired dictionary under the submenu "Euskalbar".

Options

This section contains the following items:

  • Toggle dictionaries menu

    Toggles dictionaries menu. This menu is located between "Tools" and "Help" menus in the main Firefox toolbar.

  • Preferences...

    • Default language

      Sets the language used in the language selection button when Euskalbar starts: EU > ES, ES > EU, EU > EN edo EN > EU.

    • Tab preferences

      Preferences for tab behaviour: reuse tabas and open new tabs in background.

    • Skins

      Allows to change the skin for Euskalbar interfaces.

    • Dictionaries looked up pressing Enter key

    • Combined query

      Allows to choose what dictionaries will be queried in combined queries (Shift+Enter and Ctrl+Enter). The dictionaries should be set separated by commas, without blank spaces, and using exclusively the following names:
      • Basque-Spanish-Basque translations: Euskalterm, Elhuyar, 3000, Batua, Adorez, UZEI, Mokoroa and Intza.

      • Basque-English-Basque translations: Euskalterm, Morris, Batua, Adorez, UZEI.


    You can also set the timeout to perform this queries. By default, when this queries are used, Euskalbar waits for 5 seconds to have a response. If there is no response, the connection is closed.

    This timeout use to be enough, but sometimes the servers where the dictionaries are hosted need more time to send the response.

    Warning: if the query timeout is too short, Euskalbar could receive a partial response. For instance, if you query Euskalterm with a 2 second timeout, you will get part of the whole response.

  • Statistics

    Shows a count of searches made with each of the bar dictionaries.

  • Show/hide bar buttons

    Richt-clicking in the bar shows a menu to toggle bar buttons.
