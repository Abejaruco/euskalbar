Euskalbar searches

You can look up dictionaries in several ways:

  • Pressing the toolbar buttons

    In that way, a click means a look up in the clicked dictionary.

  • Pressing the Enter key

    In that way, the dictionaries chosen in the Options section will be looked up at the same time, each one in one tab.

  • Pressing Shift+Enter keys

    If you have selected Spanish as language, Euskalbar will look up Euskalterm, Elhuyar and 3000 dictionaries, at the same time in the same tab; else, if the selected language is English, the search will be done in Euskalterm and Morris dictionaries.

  • Pressing Ctrl+Enter keys

    In that way, synonyms dictionaries will be looked up at the same time in the same tab.

  • Right-clicking

    Selecting text within a web page, doing right-click and selecting the desired dictionary under the submenu "Euskalbar.

Options

This section allows to change the following options:

  • Toggle dictionaries menu

    Toggles dictionaries menu. This menu is located between "Tolls" and "Help" menus in the main Firefox toolbar.

  • Preferences

    • Default language

      Sets the language used in the language selection button when Euskalbar starts: eu >> es, es >> eu, eu >> en edo en >> eu.

    • Tab preferences

      Preferences for tab behaviour: reuse tabas and open new tabs in background.

    • Dictionaries looked up pressing Enter key

    • Shift and Ctrl query timeout (s)

      Changes the timeout for Shift+Enter and Ctrl+Enter searches.

      When this searches are used, Euskalbar waits for 5 seconds to have a response. If there is no response, the connection is closed.

      This timeout use to be enough, but sometimes the servers where the dictionaries are hosted need more time to send the response.
