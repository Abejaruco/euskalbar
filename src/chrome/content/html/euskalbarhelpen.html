Euskalbar searches

You can look up dictionaries in several ways:
  1. Pressing the toolbar buttongs. In that way, a click means a look up in the clicked dictionary.
  2. Pressing the Enter key. In that way, the dictionaries chosen in the Options section will be looked up at the same time, each one in one tab.
  3. Pressing Shift+Enter keys. If you have selected Spanish as language, Euskalbar will look up Euskalterm, Elhuyar and 3000 dictionaries, at the same time in the same tab; else, if the selected language is English, the search will be done in Euskalterm and Morris dictionaries.
  4. Pressing Ctrl+Enter keys. In that way, synonyms dictionaries will be looked up at the same time in the same tab.
  5. Selecting text within a web page, doing right-click and selecting   "Euskaratu Opentrad erabiliz". This only works for translating Spanish texts to Basque.
