Requêtes dans Euskalbar

Les requêtes peuvent être faîtes de différentes manières:

  • En cliquant sur les boutons de la barre

    De cette manière on consulte un dictionnaire pour chaque click..

  • Avec la touche Enter

    On consulte les dictionnaires choisis dans la configuration au même temps, dans de différentes onglets.

  • Avec les touches Shift+Enter ou Ctrl+Enter

    On consulte les dictionnaires choisis dans la configuration au même temps et dans le même onglet.

  • Avec le bouton droit de la souris

    Choisir le texte d'une page web, cliquer sur le bouton droit de la souris et choisir un dictionnaire du menu contextuel.

Options

Dans cette section il y a les options suivants:

  • Montrer/cacher menu de dictionnaires

    Montre/cache le menu de dictionnaires dans la barre de menus de Firefox, entre les menus "Outils" et "Aide".

  • Configuration...

    • Langue par défaut

      Langue par défaut au lancement d'Euskalbar: EU ‣ ES, ES ‣ EU, EU ‣ FR, FR ‣ EU, EU ‣ EN, EN ‣ EU, LA ‣ EU, EU ‣ JP.

    • Configuration d'onglets

      Options pour la configuration d'onglets: réutiliser les onglets et ouvrir les onglets en arrière-plan.

    • Thèmes

      Pour changer l'aspect d'Euskalbar.

      En plus des thèmes propres d'Euskalbar, on peut utiliser un thème personnalisé. On peut télédécharger plusieurs thèmes du site web d'Euskalbar. Et en prennant ces thèmes comme base, on peut créer des nouveaux thèmes, si on connaît le format CSS.

    • Interface

      Il y a deux options pour modifier l'interface d'Euskalbar: montrer/cacher le menu de dictionnaires, eta montrer/cacher le menu contextuel.

    • Requête avancée (Enter, Shift+Enter, Ctrl+Enter)

      On peut choisir les dictionnaires à consulter dans les requêtes avancées.

    • Options de requêtes combinées

      • Configuration des requêtes combinées (Shift+Enter, Ctrl+Enter).

      Limite de temps (sec): Le comportement par dèfaut d'Euskalbar est d'attendre 5 secondes por la réponse, et si il n'y a pas de réponse dans cette temps, on ferme la connexion. Normalement cet temps est suffisant pour que le dictionnaire réponde, mais quelquefois il peut arriver que les serveurs qui hébergent les dictionnaires aient bessoin de plus de temps. Si la limite de temps est trop court, la réponse peut être coupée. Par exemple, si on consulte "eskola" dans Euskalterm avec une limite de 2 secondes, on ne reçoit pas tous les résultats.

      Voir subrequêtes: Quand on consulte Elhuyar, 3000 et le Dictionnaire Uni, avec cette option nous pouvons obtenir aussi les subrequêtes.

  • Statistiques

    Il montre le nombre de requêtes fait à chaque dictionnaire.

  • Montrer/cacher les boutons de la barre

    Si on clique le bouton droit de la souris sur la barre d'Euskalbar, on peut montrer ou cacher quelques boutons.

Touches d'accès rapide

Euskalbar offre quelques touches d'accès rapide pour faciliter le travail:
  • Ctrl+Alt+A: Montre/cache Euskalbar
  • Ctrl+Alt+S: Met le curseur á la boîte de texte et, si elle contient du texte, il est selectionné
  • Ctrl+Alt+Z: Change la direction de la traduction
  • Ctrl+Alt+H: Montre/cache le menu de dictionnaires
  • Ctrl+Alt+T: Montre/cache le menu contextuel
