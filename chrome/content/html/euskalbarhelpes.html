Consultas en Euskalbar

Las consultas se pueden realizar de diversas maneras:

  • Pulsando los botones de la barra de herramientas

    De esta manera se consulta un diccionario por vez.

  • Pulsando la tecla Enter

    De esta manera se consultan, de forma simultánea, los diccionarios marcados en el apartado "Opciones", cada uno en una pestaña.

  • Pulsando las teclas Shift+Enter ó Ctrl+Enter

    De esta manera se consultan, de forma simultánea y en la misma pestaña, los diccionarios marcados en el apartado "Opciones".

  • Pulsando el botón derecho del ratón

    Seleccionando un texto en una página web, pulsando el botón derecho del ratón y eligiendo el diccionario a consultar en el submenú "Euskalbar".

Opciones

En este apartado se pueden determinar las siguientes opciones:

  • Mostrar/esconder el menú de diccionarios

    Muestra o esconde el menú de diccionarios, situado entre las opciones "Herramientas" y "Ayuda" del menú principal de Firefox.

  • Preferencias...

    • Idioma

      Establece el idioma predeterminado del botón de selección de idioma: EU ‣ ES, ES ‣ EU, EU ‣ FR, FR ‣ EU, EU ‣ EN, EN ‣ EU ó LA ‣ EU.

    • Preferencias de las pestañas

      Establece el comportamiento de las pestañas: permite reutilizar las pestañas, y abrir las nuevas pestañas en el fondo.

    • Skins

      Permite cambiar el aspecto de Euskalbar.

      Además de los skins que incorpora Euskalbar por defecto, es posible utilizar skins personalizados. El sitio web de Euskalbar ofrece varios skins. También es posible crear nuevos skins tomando como base cualquiera de los anteriores, siempre que se conozca el formato CSS.

    • Interfaz

      Ofrece dos opciones para modificar la interfaz de Euskalbar: mostrar/esconder el menú de diccionarios, y mostrar/esconder el menú contextual.

    • Consulta avanzada (Enter, Shift+Enter, Ctrl+Enter)

      Permite seleccionar los diccionarios que se mostrarán cuando se realizan consultas avanzadas.

    • Opciones de las consultas combinadas

      Permite establecer las opciones de las consultas combinadas (Shift+Enter, Ctrl+Enter).

      Tiempo límite (seg): Por defecto, al usar las consultas combinadas Euskalbar espera respuesta durante 5 segundos. Si transcurrido este tiempo no recibe respuesta, cierra la conexión. Este periodo de tiempo suele ser suficiente para recibir respuesta de los servidores donde están alojados los diccionarios, pero en algunos casos puede ocurrir que estos servidores necesiten más tiempo. Si se establece un tiempo límite demasiado corto, puede ocurrir que se reciba solamente una parte de la respuesta. Por ejemplo, si se consulta en Euskalterm la palabra "escuela" con un tiempo límite de 2 segundos, no se reciben todos los resultados.

      Consultar subentradas: Permite consultar las subentradas de los diccionarios Elhuyar, 3000 y Batua.

  • Estadísticas

    Muestra un recuento de las búsquedas realizadas con cada uno de los diccionarios de la barra.

  • Mostrar/esconder los botones de la barra

    Pulsando con el botón derecho sobre la barra, existe la posibilidad de mostrar o esconder algunos de los diccionarios.

Atajos de teclado

Euskalbar ofrece varios atajos de teclado que facilitan el trabajo diario:
  • Ctrl+Alt+A: Muestra/esconde Euskalbar
  • Ctrl+Alt+S: Enfoca la caja de texto, y si contiene algún texto, lo selecciona
  • Ctrl+Alt+Z: Modifica el sentido del botón de selección de idioma
  • Ctrl+Alt+H: Muestra/esconde el menú de diccionarios
  • Ctrl+Alt+T: Muestra/esconde el menú contextual
